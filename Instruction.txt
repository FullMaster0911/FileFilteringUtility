Проект состоит из следующих файлов и классов:

FileFilterUtil.java: Основной класс, содержащий метод main.
FileStorage.java: Класс для хранения и обработки файлов.
FileWriter.java: Класс для записи данных в файлы.
FileStatistics.java: Класс для вывода статистики.

Версия java: 20
Версия maven: 4.0.0
Используемые библиотеки: Apache Commons CLI (

)
